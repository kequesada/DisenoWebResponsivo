Diseño Web Responsivo

El diseño web responsivo permite adaptar el diseño de una página web a diferentes tamaños de pantalla y dispositivos.

Viewport

El viewport es el área visible de una página web en un dispositivo móvil o tableta.

Media Queries

Las Media Queries permiten aplicar estilos diferentes según las características del dispositivo o la pantalla.

Bootstrap 5

Bootstrap 5 es un framework que facilita la creación de diseños responsivos y modernos.
